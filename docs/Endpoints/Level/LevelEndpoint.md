**Retornar leveis de log**
----
  Retorna todos os leveis de log existentes

* **URL**

  	/api/level

* **Method:**

	`GET`
  
*  **URL Params**

	**Required:**
 
   	Nenhum

* **Data Params**

   	Nenhum

* **Success Response:**

  * **Code:** 200 <br />
    **Content:** `{ id : 12 }`
 
* **Error Response:**

  * **Code:** 401 UNAUTHORIZED <br />  
  
  * **Code:** 403 FORBIDDEN <br />
